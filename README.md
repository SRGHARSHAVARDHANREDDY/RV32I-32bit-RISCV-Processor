# RV32I 32-bit Single-Cycle RISC-V Processor

## Overview

This project aims to design and implement a 32-bit RV32I Single-Cycle RISC-V Processor using Verilog HDL.

The processor will be built from scratch, verified through simulation, and later taken through the RTL-to-GDSII ASIC flow.

## Tools

- Verilog HDL
- Visual Studio Code
- GitHub Desktop
- Vivado
- OpenLane (Later)
